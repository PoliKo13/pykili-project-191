# Файловые статистики

Позволяет посмотреть статистику файлов компьютера, то есть общее количество файлов в папке и количество изображений, аудио, видео, документов(?), приложений(?), прочего. Возможные дополнения: отображение веса каждого из вышеперечисленных пунктов. Менее вероятные возможные дополнения: количество дублирующихся файлов.

## Подробное описание

- Должна получиться программа неизвестного пока вида
- Программа принимает на вход путь к папке

## Критерий завершенного проекта

Программа верно отображает количество файлов (+вес?) в тестовых папках с разнобразным содержимым
